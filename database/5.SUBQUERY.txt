* 서브쿼리 : 서브쿼리는 SQL문을 실행하는데 필요한 데이터를 추가로 조회하기 위해 SQL문 내부에서 사용하는 SELECT 문을 의미한다.

- 조건절에서의 서브쿼리
1. 사원 이름이 JONES인 사원의 급여 출력
SELECT 
	SAL
FROM 
	EMP
WHERE
	ENAME = 'JONES';
2. 급여가 2975보다 높은 사원 정보를 출력
SELECT 
	ENAME
FROM
	EMP
WHERE 
	SAL > 2975; 

-> 1 + 2. JONES의 급여보다 높은 급여를 받은 사원 정보 출력
SELECT
	*
FROM
	EMP
WHERE
	SAL > (select sal from emp where ename='JONES');
