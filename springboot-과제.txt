1. 목록페이지
- no, 제목, 글쓴이, 날짜
- 제목, 해당하는 내용 일부라도 검색하면 뿌려진다 -> like
2. 회원가입, 로그인 버튼이 있어야할 것.
- 회원가입
이메일, 이름, 연락처, id/pw
- 로그인
id/pw
3. 글등록 버튼이 있어야할 것.
-> 로그인이 안됐을 경우 : 로그인 페이지로 이동
-> 로그인이 됐을 경우 : 글쓰기 페이지로 이동
제목, 내용
4. 글상세
-> 제목을 누르면 상세페이지로 이동
제목, 내용, 날짜,...정보들
5. 글수정, 글삭제
-> 해당 글을 수정할 수 있는 페이지로 이동시킨다(로그인 되어 있어야할 것)
